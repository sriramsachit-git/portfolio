## üñºÔ∏è Photo Editing Recommendations

### **For Profile Photos:**

1. **Background Removal:**
   - Remove or blur background
   - Replace with gradient: `#1a1724` to `#0a0911`
   - Or use transparent PNG

2. **Color Grading:**
   - **Shadows:** Add slight purple tint (#a855f7 at 10-15% opacity)
   - **Highlights:** Add cyan/blue tint (#22d3ee at 5-10% opacity)
   - **Overall tone:** Cool, slightly purple-shifted

3. **Lighting:**
   - Increase exposure slightly (bright, modern look)
   - Add subtle rim light effect with cyan (#22d3ee)
   - Soft shadows, not harsh

4. **Effects:**
   - Slight glow/blur on edges
   - Add subtle vignette with purple tint
   - Sharpen facial features

### **For Project Screenshots:**

1. **Color Temperature:**
   - Cool white balance (slightly blue)
   - Reduce warm tones (yellows/oranges)

2. **UI Elements:**
   - If possible, tint UI elements to match:
     - Buttons: Purple (#d4b8ff)
     - Links: Cyan (#22d3ee)
     - Backgrounds: Dark (#0a0911)

3. **Borders/Frames:**
   - Add gradient border using:
     - Top-left: #d4b8ff
     - Bottom-right: #22d3ee
   - Or use transparent with slight glow
